# OpsRoute capability pack v0.1.0

This is the first supported InheritBench succession case. It binds the frozen
OpsRoute task, pinned Qwen source, pinned OLMo target, anchored transfer method,
readiness rules, and published adapter into one declarative product contract.

Hosted execution performs a deterministic verification replay. Actual model
training remains the preregistered phased CLI workflow implemented under
`inheritbench phase3b`.

The historical output schema accepted any nonempty `policy_code`. Exact scoring
still detected incorrect aliases. The registry in this pack documents the closed
vocabulary expected from future registry-backed contracts without changing any
historical score or parser behavior.
